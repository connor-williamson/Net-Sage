# NetSage

NetSage is an AI-driven network monitoring and security analysis platform.

## Vision
NetSage aims to combine:
- network telemetry collection
- systems engineering
- AI/ML-based anomaly detection
- alerting and investigation workflows

## Initial Goals
1. Build a lightweight network sensor
2. Stream telemetry to a backend service
3. Detect suspicious or unusual traffic patterns
4. Produce explainable alerts

## Planned Stack
- Go for backend/services and sensor components
- Python for ML experiments and model development
- Docker for local development
- GitHub for version control and project planning
